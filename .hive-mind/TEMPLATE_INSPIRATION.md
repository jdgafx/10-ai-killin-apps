## COMMON PREMIUM PATTERNS

**Animations:**
- Page transitions (fade/slide)
- Hover effects (scale, glow)
- Loading skeletons
- Progress indicators

**Components:**
- Toast notifications
- Modal dialogs
- Dropdown menus
- Tabs/pills navigation
- Badge indicators

**UX Patterns:**
- Keyboard shortcuts
- Undo/redo
- Auto-save indicators
- Empty states
- Error boundaries

**Tailwind Classes:**
```css
/* Glassmorphism */
backdrop-blur-lg bg-white/80 dark:bg-gray-900/80

/* Gradient text */
bg-gradient-to-r from-cyan-500 to-blue-500 bg-clip-text text-transparent

/* Shadow glow */
shadow-lg shadow-cyan-500/50

/* Smooth transitions */
transition-all duration-300 ease-in-out

/* Hover effects */
hover:scale-105 hover:shadow-xl
```
